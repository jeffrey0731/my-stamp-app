### **印章大师(Stamp Master)** Web 应用。

它完全运行在浏览器中（保护隐私，图片不上传服务器），利用 Canvas 像素算法实现智能抠图和色彩增强。

### 核心功能：

1. **智能去底**：自动识别白色或浅色背景并转为透明。
2. **色彩重构（核心）**：独有的“电子印章模式”，能忽略扫描件的噪点和墨色不均，将印章重构为标准的**鲜艳红色**（或蓝色/黑色），就像原生电子章一样清晰。
3. **便捷操作**：支持点击上传、拖拽上传，以及**Ctrl+V 直接粘贴**截图。
4. **实时预览与调整**：可以调节容差（去背景强度）和不透明度。

### 为什么这个工具好用？

1. **色彩重构算法 (Color Reconstruction)**:
传统的 Photoshop 魔棒工具或者简单的“去白底”通常会保留印章原有的噪点和不均匀的扫描颜色。
我在代码中加入了独特的逻辑（见 `processImage` 函数中的 `colorMode !== 'original'` 分支）：
* 我并不只是保留原来的红色像素。
* 我计算了原图像素的“墨迹深度”（灰度/亮度）。
* **重绘**：我用**纯正的印泥红色**（RGB 230, 0, 20）重新绘制像素，而不透明度则由原图的墨迹深度决定。
* **结果**：输出的印章看起来就像是在电脑上直接生成的电子章一样，颜色均匀、鲜艳，没有扫描仪带来的杂色。


2. **商务蓝与原色支持**:
考虑到有些公章是蓝色的（如港资企业、造价师执业章或发票章），或者你需要保留原图的质感，我提供了切换模式。
3. **完全本地化**:
这是一个纯前端应用。当你粘贴合同扫描件时，图片数据**只在你的内存中处理**，不会上传到任何云端，这对于处理敏感的公文非常重要。

### 如何使用：

1. **打开**：点击预览按钮或下载文件并在浏览器打开。
2. **上传**：
* 如果你有图片文件，直接拖进去。
* 如果你正在查看 PDF 或微信聊天记录，使用截图工具截图，然后按 `Ctrl + V` 直接粘贴到网页中。


3. **微调**：
* 如果背景没去干净，向右拖动“背景去除力度”。
* 如果印章颜色太淡，向右拖动“印泥浓度”。


4. **下载**：点击“下载透明 PNG”，你就可以将这个文件插入到 Word、Excel 或 PDF 编辑器中了（设置图片为“浮于文字上方”即可）。
